In&Out部分：在文本框里输入内容在点击下面四个按钮可以分别改变一些值。
set star size：设置手动加入的星球的大小。
set star dir:在加入卫星模式下，设置卫星的旋转方向（顺时针或逆时针）
set tracklen: 设置轨迹长度
set Gravity:设置重力常数

控制（control）部分：
Stop/Start 开始或者停止模拟
show infs/unshow infs：显示或不显示星球信息
Clear all：删除所有星球
Set merge：设置星球是否可以合并

模式选择（modes select）部分：
Show inf：该模式下点击星球可以显示星球的信息
clear star：该模式下可以通过点击星球来删除星球
add mood：在该模式下，先通过点击选中一个星球作为母星，再在空白处点击，会生成大小为starsize的，顺/逆时针围绕着母星旋转的卫星
add sun：在该模式下，点击空白的地方来加入恒星，恒星不会运动，不会与其他星球融合
add star：通过鼠标来添加行星，可以通过拖动来控制初始速度，行星大小为starsize

数据存取（SL data）部分：
Get seed：获得目前的星球数据，输出到文本框中
Load seed：载入文本框中的星球数据。
